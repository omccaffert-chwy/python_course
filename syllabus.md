

# Course Syllabus

Welcome to the Python Course! Below is the schedule outlining what to complete and when. Each day includes ~2 hours of work: Udemy lectures (optional if you know the matieral already) + completing the project in the corresponding `day_X` folder.

## Getting Started Checklist (Day 1)

- [ ] Accept the GitHub repo invitation (check your Chewy email)
- [ ] Read `README`
- [ ] Read `git.md` to learn how to fork the repo
- [ ] Open `day_01/README.md` and complete the Udemy lectures (if needed)
- [ ] Complete `day_01/project.py` and run the tests
- [ ] Commit your work to a new branch and submit a PR for teammate review

**Office Hours:** Tuesday and Thursdays 2:30 PM - 3:30 PM (available for questions and walkthroughs)
